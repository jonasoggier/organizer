== Welcome to TheNextWeek

TheNextWeek was pair programmed by Samir Remtulla (samirremtulla) and Jonas Oggier during the Bitmaker Hackathon having taken place on April 13-14, 2013.

TheNextWeek is a simple program and week planer for Bitmaker Labs, enabling (i) students to access imporant information regarding the program and (ii) teachers to continously upload new content.
